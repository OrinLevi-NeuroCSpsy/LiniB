## נושאים עיקריים

### לינארית 1ב

| נושא | תיאור |
|------|-------|
| **מערכות משוואות לינאריות** | דירוג מטריצות, צורה קנונית, מטריצות אלמנטריות |
| **מרחבים וקטוריים** | הגדרות, תתי-מרחבים, סכום ישר |
| **תלות ואי-תלות לינארית** | קבוצות בלתי תלויות, בסיס ומימד |
| **מרחבי שורות ועמודות** | מרחב שורות, מרחב עמודות, דרגה |
| **העתקות לינאריות** | הגדרה, גרעין ותמונה, מטריצה מייצגת |
| **דטרמיננטה** | מינור, מטריצה מצורפת, פיתוח לפלס |
| **כלל קרמר** | פתרון מערכות באמצעות דטרמיננטות |
