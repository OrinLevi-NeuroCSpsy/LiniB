# תרגול - לינארית 2ב

!!! info "בקרוב..."
    עמוד זה יכלול שאלות תרגול לקורס לינארית 2ב.

    **נושאים שיכללו:**

    - ערכים עצמיים ווקטורים עצמיים
    - לכסון מטריצות
    - מכפלה פנימית ומרחבי מכפלה פנימית
    - תהליך גרם-שמידט
    - הטלות אורתוגונליות
    - אופרטורים נורמליים וצמודים לעצמם
    - משפט הספקטרום

---

## מבנה הקורס

<div class="definition-box" markdown>

**לינארית 2ב** מרחיבה את החומר של לינארית 1ב ומתמקדת ב:

1. **ערכים עצמיים ולכסון** - מציאת צורה אלכסונית למטריצות
2. **מכפלה פנימית** - גיאומטריה במרחבים וקטוריים
3. **מרחבים אורתוגונליים** - בסיסים אורתונורמליים
4. **אופרטורים מיוחדים** - סימטריים, אורתוגונליים, ועוד

</div>
